##	跨平台多端广东轻院生活开发

		广东轻院生活是一款针对移动端，致力于方便校园师生（特别新生）了解广东轻院各校区生活的Web应用程序。对校区内的各种生活设施进行详细介绍，方便学生尽快融入学校生活。`了解广东各食堂的特色档口、各种体育设施位置及使用介绍、各教学实训楼及教室位置、收取快递、看病、失物招领等`。特有的点评推荐功能便于师生快速熟悉学校生活的方方面面。

1. 食堂各档口经典美食推荐及点评
2. 各生活设置的导航功能
3. 校园热点信息的生成及推荐

## 项目需求

一、**前端界面开发(移动端)**

LoGO 设计、登录注册界面、首页、美食推荐、评论、导航等技术

**技术:**

- uni-app u-view框架  color-UI框架

**二、后台管理开发**

用户管理(学生、教师、管理员)，档口管理、餐品管理、教室管理、生活设施管理等

**技术:**

- node.js koa2 阿里云云数据库
